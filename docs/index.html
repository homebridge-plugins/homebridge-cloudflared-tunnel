Homebridge Cloudflared Tunnel

The Homebridge Cloudflared Tunnel plugin allows you to run a Cloudflare-Tunnel for exposing your Homebridge instance for remote access.

Installation

  1. Search for "Cloudflared Tunnel" on the plugin screen of Homebridge UI.
  2. Click Install.
  3. Set Auto Install of Cloudflared Tunnel in Plugin Configs.
  4. Restart Plugin and Cloudlfare URL will be displayed in logs.
